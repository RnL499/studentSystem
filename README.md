## 快速安裝與啟動

### 1. 建立並啟用虛擬環境

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

### 2. 安裝相依套件

```powershell
pip install -r requirements.txt
```

### 3. 資料庫初始化與遷移

```powershell
python manage.py makemigrations
python manage.py migrate
```

### 4. 建立超級管理員帳號

```powershell
python manage.py createsuperuser
```

### 5. 建立 media 資料夾（用於頭像上傳）

```powershell
mkdir media
mkdir media\avatars
```

### 6. 啟動開發伺服器

```powershell
python manage.py runserver
```

瀏覽器開啟 `http://127.0.0.1:8000/login/` 即可開始使用。

## 功能操作說明

### 一、帳號與使用者管理 (A01)

#### 學生自行註冊
1. 進入登入頁，點選「註冊帳號」
2. 輸入帳號、密碼（需符合強度規範），點選「註冊」
3. 註冊成功後自動登入

#### 管理員建立教師
1. 左側導覽列 → 「教師管理」
2. 填寫教師帳號、密碼、所屬系所
3. 點選「建立教師帳號」

#### 管理員帳號與權限管理
1. 左側導覽列 → 「帳號與權限管理」
2. 可搜尋所有使用者，查看帳號、姓名、Email、目前角色
3. 可變更使用者角色（學生 / 教師 / 管理員）
4. 可刪除使用者帳號

#### 修改個人資料
- 所有角色皆可透過側邊欄的「修改個人資訊」修改姓名、Email、頭像

---

### 二、課程管理 (A02–A04)

#### A02 — 新增課程
- **管理員**：左側導覽列 → 「課程管理」→ 填寫課程代碼、名稱、描述、上課時間、學分數、人數上限、授課教師 → 「確認建立課程」
- **教師**：左側導覽列 → 「新增課程」→ 同上（教師欄位自動鎖定為自己）
- 課程時間採用**圖形化多選元件**：選擇星期幾 × 第幾節，支援複選多個時段

#### A03 — 修改課程
- **管理員**：在課程管理頁的課程列表，點選「編輯」按鈕修改
- **教師**：在修課名單頁，點選自己課程的「編輯」按鈕
- 可修改所有課程欄位（課程名稱、描述、時間、學分、人數上限）
- 管理員可額外重新指派授課教師

#### A04 — 刪除課程
- **管理員**：在課程管理頁的課程列表，點選「刪除」按鈕
- 教師無法自行刪除課程，需請求管理員協助

---

### 三、選課管理 (A05–A08)

#### A05 — 審核選課結果
- 教師 / 管理員：在修課名單頁面，可看到待審核的選課申請
- 點選「核准」或「駁回」按鈕進行審核

#### A06 — 查詢已選課程
- 學生登入後 →「我的修習課程」查看所有已選修課程列表

#### A07 — 加選課程
- 學生 → 「可加選課程」→ 在可選課程中點「加選」
- 系統會自動檢查：
  - 是否在選課開放期間
  - 課程人數是否已達上限
  - 是否已選修同一門課

#### A08 — 退選課程
- 學生 → 「選退課管理」→ 找到已選課程，點「退選」
- 需在退選期間開放時才可操作

> **管理員控制**：在管理儀表板的「系統控制面板」可一鍵開啟 / 關閉選課或退選期間。

---

### 四、修課學生名單 (A09–A10)

#### A09 — 查詢修課學生名單
- 教師 →「修課名單」→ 選擇課程 → 查看所有修課學生
- 管理員 → 課程管理中亦可查看

#### A10 — 匯出修課學生名單
- 在修課名單頁面，點選「匯出學生名單」按鈕
- 系統產生 **CSV 格式檔案**下載（含 BOM 以相容 Excel 中文）

---

### 五、成績管理 (A11–A14)

#### A11 — 教師輸入成績
- 教師 → 「成績管理」→ 選擇課程 → 看到所有學生
- 可輸入每位學生的期中成績與期末成績

#### A12 — 教師匯出成績
- 在成績管理頁面，點選「匯出成績」
- 匯出含期中、期末、平均成績的 **CSV 報表**

#### A13 — 教師修改成績
- 在成績管理頁面直接修改分數，點擊「儲存」

#### A14 — 成績總覽
- 教師 → 「成績管理」→ 「總覽」
- 以全覽表格呈現所有課程的學生成績（期中、期末、平均）
- 可匯出全部成績

---

### 六、成績查詢與報表 (A15–A16)

#### A15 — 學生查詢成績
- 學生 → 「我的修習課程」→ 可查看各科期中 / 期末成績與學期平均
- 成績平均採用**學分加權平均**計算：
  $$\text{學期總平均} = \frac{\sum (\text{各科平均} \times \text{學分數})}{\sum \text{學分數}}$$

#### A16 — 成績報表（列印 / PDF）
- 學生 → 「成績查詢」→ 可透過下拉選單選擇學期
- 點選「🖨 列印成績單」按鈕
- 列印時自動隱藏側邊欄、按鈕等 UI 元素，只保留乾淨排版的成績報表
- 可直接列印或儲存為 PDF

---

### 七、留言系統

- 登入使用者可在課程頁面新增留言
- 使用者僅能編輯自己的留言
- 管理員可刪除任何留言

---

### 八、課表查詢

- 學生 → 「學生課表」
- 以週一至週五 × 第 1～8 節的**表格化課表**顯示
- 每個格子包含課程名稱與教師名稱
- 可列印課表（自動隱藏非課表元素）
- 可匯出 CSV 課表

---

### 九、系統管理（管理員專屬）

#### 系統控制面板
- 位於「管理儀表板」頁面
- **選課期間開關**：一鍵切換選課是否開放
- **退選期間開關**：一鍵切換退選是否開放
- **資料備份**：點選「💾 下載資料庫備份」即可下載 SQLite 資料庫副本

#### 踢出學生 / 刪除帳號
- 在課程管理頁面可將學生移出課程
- 在「帳號與權限管理」可刪除任何非自己的帳號

## 注意事項

1. 登入後系統會根據使用者角色自動導向對應的儀表板（管理員 → 管理儀表板、教師 → 教師儀表板、學生 → 學生儀表板）。
2. 若有既有資料庫，更新模型後請先備份 `db.sqlite3` 再執行 `migrate`。
3. 範本中的角色檢查同時相容 `Profile.is_teacher` 標記與 `Teacher` 群組。
4. 系統預設使用開發用 `SECRET_KEY`，部署至正式環境前請務必更換。
